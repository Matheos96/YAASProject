Our site has an REST API Available for use. You can use the web interface available at /rest/auctions or any other client.

For all users
To get all auctions send a GET request to /rest/auctions
To search for an auctions send a GET request to /rest/auctions?q=searchword
To get a specific auction send a GET request to /rest/auctions/id where id is the auction id

For Registered Users Only
You have to specify your login credentials in your request (Basic Authentication)

To bid on an auction, first GET the specific auction using /rest/auction/id where id is the auction id,
then send a POST request to the same URL with json content {"bid": amount} attached, where amount is an integer or decimal number (Only two decimal points will be considered)
